# Installation
<html>

</head>
<body>
  <div id="player"></div>
  <script src="asciinema-player.min.js"></script>
  <script>
    AsciinemaPlayer.create(
      '485034.cast',
      document.getElementById('player'),
      { cols: 94, rows: 28 }
    );
  </script>
</body>

npm install -g picloud-cli
