# Getting started
## Installation

``` 
npm install -g picloud-cli 
```

## Quick Start

> `macOS`, `Ubuntu`, `Window-beta` 

1. 터미널 실행
1. 이미지 폴더로 이동
1. `picloud` 입력
1. 회원가입 진행 후 이미지 업로드

### Feature

 - cli 명령어로 손쉽게 이미지를 관리할 수 있습니다.
 - 대량의 이미지를 저장하시는 분들을 위해 편리하게 만들었습니다.
 - 나만의 이미지 저장소를 가질 수 있습니다.
